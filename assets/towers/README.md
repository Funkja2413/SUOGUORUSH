# Tower Asset Slots

Replace files in these fixed paths to update tower visuals without changing config.

Static assets:
- Single PNG frame.
- Transparent background.
- One frame should be 4:3.

Idle spritesheets:
- PNG spritesheet.
- 4 columns x 3 rows.
- 12 frames total.
- Each frame should be 4:3.
- Current default playback is 4fps.
